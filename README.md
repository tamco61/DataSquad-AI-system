Проект обернут в докер контейнер собирается при помощи консольной команды:

```docker-compose up —build```

# Взаимодействие пользователя с ИИ-ассистентом
После получения корпоративного письма на почту, автоматически происходит зазгрузка текста письма в систему. Производится комплексная обработка письма (подробнее в сервисах). ИИ-асистент генерирует ответное письмо, соблюдая требования: оформление по брендбуку, сохранение корпоративного стиля письма, проверка пунктуации и смысла. Далее письмо прикрепляется на Dashboard, пользователю приходит уведомление о добавлении нового письма. Процесс согласования является обязательным, без него письмо не отправится.

# Основные сервисы
**1. Ingestion service**

Сервис принимает входящие письма, извлекает из них текст, адрес эл. почты отправителя, вложения, запускает первичную обработку.

Функции:

-Загрузка писем

-OCR

-Сохранение оригиналов

-Отправка события новое письмо в очередь


**2. AI Analysis Service**

Использует LLM для разбора письма и формирования структуры.

Функции:

-Классификация письма

-Извлечение сути запроса

-Определение срочности

-Определение уровня формальности

-Выявление юридических рисков

-Рекомендации по маршрутизации


**3. Workflow Service**

Ядро всей системы — хранит карточки писем и управляет их статусами.

Функции:

-Создание карточки письма

-Маршрутизация по отделам

-Статусы (в обработке, на согласовании, готово и т. д.)

-Хранение участников

-SLA-таймеры


**4. Draft generation service**

Создаёт варианты ответов на основе анализа письма.

Функции:

-Официальный стиль

-Корпоративный стиль

-Клиентский стиль

-Краткий ответ

-Юридическая само-проверка текста


**5. Approval Service**

Организует процесс согласования между отделами.

Функции:

-Комментирование

-Отслеживание правок

-Подсветка спорных формулировок (AI)

-История согласований

-Финальное подтверждение отправки



**6. Notification Service**

Отправляет уведомления сотрудникам и следит за дедлайнами.

Функции:

-email / Telegram / Slack уведомления

-Напоминание о дедлайнах

-Эскалация при нарушении SLA

-Уведомления о новых задачах и согласованиях


**7. Analytic Service**

Собирает статистику для руководства и аналитики.

Функции:

-Среднее время ответа

-Соблюдение SLA

-Типы писем

-Загрузка подразделений

-Дэшборды
