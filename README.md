# MVP решение
## Описание
Реализована бизнеc-логика работы с LLM моделями (Анализатор сообщения и генератор ответа)
Для MVP бизнес-логику с разных сервисов перенесли в один для простоты развёртки.

## Инструкция по эксплуатации
Установка зависимостей
```pip install -r requirements.txt```
Заполнение
.env
```folder_id, api_key```
Запуск сервиса из папки mvp
```uvicorn app.main:app --reload --host 0.0.0.0 --port 8001```
